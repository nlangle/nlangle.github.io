Some convenient shortcuts for Atom.io 
⌘ = command
⇧ = shift
⌃ = control
⌥ = alt / option

⌘ + L = select line
⌥ + B = move to beginning of word
⌥ + F = move to end of word
⌥ + ⇧ + B = select to beginning of word
⌥ + ⇧ + F = select to end of word

⌘ + ⇧ + D = duplicate line
⌘ + ⇧ + K = delete line

⌘ + D = match characters
⌘ + U = unmatch characters
⌘ + ⌃ + G = select all matching characters

⌃ + M = move to matching bracket
⌘ + ⌃ + M = select all inside matching brackets

⌘ + [ or ⌘ + ] = indent/outdent selected lines
⌘ + ⌃ + up/down = move line up/down

⌘ + / = toggle comments
⌘ + click = apply multiple cursors

⌘ + F = find in current
⌘ + ⇧ + F = find in project

⌃ + ⌥ + T = toggle split diff

select all -> edit -> lines -> auto indent = auto indent all code
