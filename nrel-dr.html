Data Repository (DR) Projects

This is a brief tutorial about getting a DR project ready for a local environment.

  1. Install ElasticSearch

    http://stackoverflow.com/questions/23034863/install-elasticsearch-1-1-using-brew

  2. git

    git checkout/pull latest from repository

  3. build database

    start mongo

    mongo
    mongod

    build/populate/index mongo database with data

    set up mongo

    index

    cd opt_bin/

    run script

    ./xdr_elasticsearch_indexer_cli.php

  4. build symlink

    need to build a symlink for local.settings.php

    cd /Users/nlangle/Sites/nreldr/
    cd assets/
    ln -s localSettings.php.localdev localSettings.php
    ls -al

    look for:

    lrwxr-xr-x   1 nlangle  staff     26 Feb 11 09:41 localSettings.php -> localSettings.php.localdev

  5. enable elasticsearch

    /Users/nlangle/Sites/nreldr/assets/localSettings.php.dev

    set useElasticsearch = true

    const useElasticsearch = true;

  6. Start Elasticsearch

    open terminal, run:

    elasticsearch

    to verify it is running, open in web browser:

    http://localhost:9200/

    it is VERY memeory intensive, so be sure to turn it off when not needed

  7. Stop Elasticsearch

    When not using for local development, disable elasticsearch

    from the terminal window that started elasticsearch

    cntrl c

    OR in a new terminal shell:

    pkill -f elasticsearch

    Then, disable in the project:

    /Users/nlangle/Sites/nreldr/assets/localSettings.php.dev
    const useElasticsearch = false;
