Quick way to integrate your GA in Google Sheets

  1. Open new Google Sheet
  2. Click Add-ons
  3. Select Google Analytics
  4. Create New Report
  5. Enter info in right sidebar
  6. Create report

Running reports

  • To run all the reports you've created select "Add-ons" > "Google Analytics" > "Run Reports" from the menu bar.
